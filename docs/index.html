Политика конфиденциальности XCurrency

Дата вступления в силу: 15.12.2025

XCurrency — это простое приложение-калькулятор валют. Мы уважаем вашу приватность и сознательно сделали приложение таким, чтобы не собирать персональные данные пользователей.

1. Какие данные мы собираем

Мы не собираем, не храним и не передаём:

  • персональные данные (имя, email, номер телефона и т.п.);
  • идентификаторы устройства (IDFA, IDFV) для отслеживания;
  • геолокацию;
  • контакты, фотографии, файлы;
  • историю действий пользователя;
  • данные о покупках (покупки в приложении отсутствуют).

2. Как работает приложение

Приложение XCurrency предназначено для конвертации валют.

  • Курсы валют загружаются из открытых публичных источников.
  • Все курсы приводятся через базовую валюту — доллар США (USD).
  • Конвертация валют выполняется локально на устройстве пользователя.

Мы не получаем и не сохраняем значения, которые пользователь вводит в калькулятор.

3. Источники курсов валют

Для получения информации о курсах валют приложение использует данные из следующих открытых источников:

  • exchangerate.host
  • frankfurter.app
  • open.er-api.com
  • api.coingecko.com

Данные используются исключительно для отображения актуальных курсов и выполнения расчётов.

4. Сторонние сервисы

XCurrency не использует:

  • рекламные SDK;
  • аналитику и трекинг поведения пользователей;
  • системы профилирования;
  • регистрацию и авторизацию пользователей;
  • интеграции с социальными сетями.

При обращении к внешним источникам курсов валют такие сервисы могут технически получать стандартные сетевые данные (например, IP-адрес и user-agent), что является обычным свойством интернет-соединений и не контролируется приложением.

5. Разрешения

Приложение не запрашивает доступ к данным, не связанным с его основной функцией.

XCurrency использует доступ к интернету исключительно для загрузки курсов валют из открытых источников.

6. Дети

Приложение имеет возрастной рейтинг 4+ и подходит для широкой аудитории. Мы сознательно не собираем персональные данные пользователей, включая данные детей.

7. Безопасность

Мы не храним пользовательские данные на серверах, так как не собираем их. Рекомендуется использовать актуальную версию iOS для обеспечения безопасности сетевых соединений.

8. Изменения политики

Мы можем обновлять настоящую Политику конфиденциальности в случае изменения функциональности приложения или требований платформ. Обновлённая версия вступает в силу с момента публикации.
